## ✨ 特性

- 📦 支持深色浅色模式。
- 🛡 使用 TypeScript 开发，提供完整的类型定义文件。

## 🖥 兼容环境

- 服务端渲染。
- 目前只测试过`chrome`

## 📦 安装

```bash
npm install @oc/oc --save
pnpm add @oc/oc
```

## 本地调试指南

> install

1. `pnpm link --global`
2. 目标项目执行`pnpm link @oc/design --global`

> uninstall

1. 目标项目执行`pnpm unlink @oc/design`
2. `pnpm remove @oc/design --global`

## 本地实时预览

`pnpm run doc:dev`
